To run
1. install required packages via pip (see Packages.txt)
2. cd into the folder containing chat_bot.py
3. run the command python3 chat_bot.py
4. Login to https://web.libera.chat/ with the channel CSC482
5. Interact with the bot like this: "pog-bot: [YOUR MESSAGE/COMMAND]

For phase_3/jeremiah_bot/jeremiah_bot.py:
LibreTranslate runs locally on http://localhost:5000
To change this, simply edit the api_url on line 12 of jeremiah_bot.py
